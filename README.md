## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- MongoDB database
- API keys for: Gemini, Resend, Inngest, Arcjet

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/welthos.git
   cd welthos
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment variables**
   
   Create a `.env` file:
   ```env
   DATABASE_URL=mongodb+srv://...
   JWT_SECRET=your-super-secret-key
   RESEND_API_KEY=re_...
   GEMINI_API_KEY=AI...
   INNGEST_EVENT_KEY=...
   INNGEST_SIGNING_KEY=...
   ARCJET_KEY=ajkey_...
   ```

4. **Generate Prisma client**
   ```bash
   npx prisma generate
   npx prisma db push
   ```

5. **Start development server**
   ```bash
   npm run dev
   ```

6. **Open [http://localhost:3000](http://localhost:3000)**

## 🗄️ Database Schema

```prisma
model User {
  id                 String    @id @default(auto()) @map("_id") @db.ObjectId
  email              String    @unique
  password           String
  name               String
  imageUrl           String?
  resetToken         String?
  resetTokenExpiry   DateTime?
  transactions       Transaction[]
  accounts           Account[]
  budgets            Budget[]
}

model Account {
  id           String        @id @default(auto()) @map("_id") @db.ObjectId
  name         String
  type         AccountType   // CURRENT | SAVINGS
  balance      Float         @default(0)
  isDefault    Boolean       @default(false)
  userId       String        @db.ObjectId
  transactions Transaction[]
}

model Transaction {
  id                String             @id @default(auto()) @map("_id") @db.ObjectId
  type              TransactionType    // INCOME | EXPENSE
  amount            Float
  description       String
  date              DateTime
  category          String
  receiptUrl        String?
  isRecurring       Boolean            @default(false)
  recurringInterval RecurringInterval? // DAILY | WEEKLY | MONTHLY | YEARLY
  status            TransactionStatus  @default(COMPLETED)
  userId            String             @db.ObjectId
  accountId         String             @db.ObjectId
}

model Budget {
  id            String    @id @default(auto()) @map("_id") @db.ObjectId
  amount        Float
  lastAlertSent DateTime?
  userId        String    @unique @db.ObjectId
}
```

## 🤖 Background Jobs (Inngest)

| Job | Trigger | Description |
|-----|---------|-------------|
| `process-recurring-transaction` | Event | Process individual recurring transactions |
| `trigger-recurring-transactions` | Cron (daily) | Find and trigger due recurring transactions |
| `generate-monthly-report` | Cron (monthly) | Generate and email financial reports |
| `check-budget-alerts` | Cron | Send email alerts when budget exceeds 80% |
